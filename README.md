# 統合業務ツール（Chrome拡張機能）
## 解決した課題
| Before | After |
|--------|-------|
| 部署内の異なるチーム（プロダクト）ごとにマニュアルやツールが乱立 | 1つの拡張に統合 |
| 拡張機能を活用したいが数が増えると切り替えにも手間がかかってしまう | メニューからスムーズに選択できる |

## 機能について
- まったく異なる機能を一つの拡張機能内で利用可能
- トップ画面からチーム（プロダクト）を切り替えて、それぞれ異なるメニューの利用が可能
- 今後ひな型に倣って機能を追加可能

## 実装したツールの例
### ナレッジベース
- カテゴリ階層、検索、Markdown対応
- スプレッドシートをデータベースとするため誰でも追加・編集が可能
- 属人化を防止および年 **40時間/人** ほどの時間短縮

### 顧客審査補助ツール
- 審査フローや結果を入力するためのテキスト自動生成機能
- ミスの防止および年 **20時間/人** ほどの時間短縮

### Zendesk用FAQエディター
- ハイライトやカードの挿入等、HTMLタグを直感的に自動生成する機能
- 誰が作ってもデザインが統一される
- HTMLタグを都度入力するのと比べて1件あたり **10分** ほどの短縮
- 新規事業の開始に伴い数百件のFAQを作成する必要があったため実装

# Zendeskデータエクスポーター（Python / GUI）
## 解決した課題
| Before | After |
|--------|-------|
| 毎週手動でZendesk管理画面からデータをエクスポート | ワンクリックまたは自動で出力 |
| 複数ブランドのデータを個別に取得して手作業で統合 | ブランド選択して一括エクスポート |
| 日別・ブランド別の統計を手計算 | 問い合わせ件数・UUを自動集計、KPIへの転記 |
| 別プロダクト用のZendeskでは使用できない | UI上で認証してどのZendeskでも汎用的に使用可能 |

## 機能について
- Zendesk APIからチケットデータを取得し、Excel形式でエクスポート
- 認証情報の暗号化保存
- ブランド / カラム / 除外タグを選択可能
- AIエージェント対応チケットを別シートに分離するオプション
- 日別統計シート（UU数・回答数）の自動生成
- タスクスケジューラによる週次自動実行対応
- 古いエクスポートファイルの自動アーカイブ（14日経過でoldフォルダへ）
- レポート作成の完全自動化により **30分/週** の作業を削減
